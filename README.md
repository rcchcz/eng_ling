## Nomenclatura de variáveis
- Case sensitive
- Definição de nomes
  - É permitido letras, dígitos e underscore.
  - Deve iniciar com letra.
  - Não é permitido caracteres especiais e espaços em branco. 
- Snake case
- Tudo maiúsculo para constantes
